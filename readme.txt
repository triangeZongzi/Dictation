1. 文件打开：左上角file菜单中的open选项，选择你存储的单词文件路径
2. 听写：载入文件后根据新的弹窗提示在文本框中输入单词
3. 音频：部分单词包含美式发音，大部分没有，会在后续版本更新
目前只支持
W5D5 
W6D1 W6D2 W6D3 W6D4 W6D5
W7D1 W7D2 W7D3 W7D4 W7D5

bug1（windows的命令不能使用utf-16的方式进行编码）
playsound中55行：
将源码中command = ' '.join(command).encode('utf-16')变为command = ' '.join(command)即可
